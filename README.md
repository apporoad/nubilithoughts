日月同辉，地广天空

皆即风格，响彻寰宇

上可通天，下可遁地

世间万物，均循其理

中外语言，各应其灵

退散霸各，炸天性能

先于他人，功能速出

敏捷无限，无人其右

有始有终，一片祥和
